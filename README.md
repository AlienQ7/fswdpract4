# 📸 Dynamic Image Gallery (JavaScript + API) #
web_demo : https://fswdpract4.pages.dev

## 🚀 Features ##

- 🔄 Fetch images dynamically from API
- 🖼️ Responsive grid layout (Bootstrap)
- 🎯 Clean UI with overlay (title + date)
- 🔍 Click to view full image (custom modal)
- 🎨 Modern gradient background
- ⚡ Lightweight (no Bootstrap JS required)


## 🛠️ Technologies Used ##

- HTML5
- CSS3 (with CSS Variables / Root Variables)
- JavaScript (Fetch API)
- Bootstrap 5 (Grid system only)
